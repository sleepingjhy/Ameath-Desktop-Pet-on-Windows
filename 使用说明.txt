Ameath Desktop Pet（Windows）使用说明

一、快速开始
1. 双击根目录中的 AmeathDesktopPet.exe 启动程序。
2. 启动后会同时出现：
   - 桌宠窗口（可拖动、右键操作）
   - 应用主界面（设置 / 音乐 / 关于）
3. 如果关闭主界面，程序仍可能在系统托盘运行，可从托盘重新打开。

二、音乐文件位置（可自行添加）
1. 音乐文件夹在程序根目录的 music 文件夹中。
2. 支持常见音频格式：.ogg / .mp3 / .wav / .flac / .m4a。
3. 你可以直接把新音乐文件复制到 music 文件夹，然后在应用中播放。
4. 也可以在应用“音乐”页面使用“加入本地歌曲”按钮导入。

三、桌宠基础操作
1. 左键按住桌宠：可拖动位置。
2. 右键桌宠：打开菜单，可设置
   - 打开应用界面（快速打开设置/音乐/关于主窗口）
   - 停止移动 / 恢复移动
   - 跟随鼠标
   - 缩放比例
   - 透明度
   - 显示优先级
   - 多开数量
   - 开机自启
   - 关闭桌宠
3. 右键时，当前操作的桌宠会出现天蓝色描边提示。

四、音乐功能说明
1. 右键桌宠 → 音乐：可快速上一首 / 播放暂停 / 下一首、改播放模式、调音量。
2. 应用“音乐”页提供完整功能：
   - 点击歌曲直接播放
   - 拖拽调整播放顺序
   - 调整音量与播放模式
   - 右键歌曲可重命名（会同步修改 music 文件夹中的文件名）

五、批量删除歌曲（音乐页）
1. 点击“删除列表歌曲”进入删除模式。
2. 按提示“已进入删除模式，请勾选歌曲后确认”，在歌曲右侧方框勾选多首。
3. 点击“确认删除”后可选择：
   - 仅从列表移除
   - 从列表移除并删除本地文件
4. 点击“取消删除”可退出删除模式。

六、托盘与关闭行为
1. 关闭主窗口时，可能按设置最小化到托盘而不是退出。
2. 托盘图标右键可“打开”或“退出”。
3. 如需彻底退出，请使用托盘“退出”或应用内退出按钮。

七、常见问题
1. 没有声音：
   - 先检查应用内音量是否为 0
   - 检查系统输出设备是否正确
   - 确认 music 文件夹中有可播放音频文件
2. 新歌曲没显示：
   - 确认文件扩展名受支持
   - 重新打开音乐页面或重启应用
3. 无法删除文件：
   - 可能文件被其他程序占用，请关闭占用程序后重试

八、升级与数据
1. 配置会保存在系统用户目录中，重启后会保留。
2. 建议定期备份 music 文件夹中的自定义歌曲。

祝你使用愉快！

Ameath Desktop Pet (Windows) User Guide

1. Quick Start
1) Double-click AmeathDesktopPet.exe in the root folder to launch.
2) After startup, you will see both:
   - Desktop pet window (draggable, right-click menu)
   - Main app window (Settings / Music / About)
3) If you close the main window, the app may still run in the system tray.

2. Music Folder (you can add files manually)
1) Music files are stored in the music folder under the app root directory.
2) Supported formats: .ogg / .mp3 / .wav / .flac / .m4a.
3) You can copy music files directly into the music folder.
4) You can also import tracks from the Music page using “Add Local Tracks”.

3. Basic Desktop Pet Operations
1) Left-click and hold the pet to drag it.
2) Right-click the pet to open the menu. Available options include:
   - Open app window (quick access to Settings/Music/About)
   - Stop Moving / Resume Moving
   - Follow Mouse
   - Scale
   - Opacity
   - Display Priority
   - Instance Count
   - Launch at Startup
   - Close Pet
3) While right-click menu is open, the active pet shows a sky-blue outline.

4. Music Features
1) Right-click pet -> Music: quick control for Previous / Play-Pause / Next,
   play mode switching, and volume.
2) The Music page provides full controls:
   - Click a track to play
   - Drag to reorder playlist
   - Adjust volume and play mode
   - Right-click track to rename (also renames the file in the music folder)

5. Batch Delete Tracks (Music Page)
1) Click “Delete Tracks” to enter delete mode.
2) Follow the prompt and check multiple tracks in the right-side checkboxes.
3) Click “Confirm Delete”, then choose:
   - Remove from playlist only
   - Remove from playlist and delete local files
4) Click “Cancel Delete” to exit delete mode.

6. Tray and Close Behavior
1) Closing the main window may minimize to tray (depending on settings).
2) Right-click the tray icon for “Open” or “Exit”.
3) To fully quit, use tray “Exit” or the in-app exit button.

7. FAQ
1) No sound:
   - Check if app volume is set to 0
   - Check system output device
   - Ensure playable audio files exist in the music folder
2) New tracks do not appear:
   - Verify file extension is supported
   - Reopen the Music page or restart the app
3) Cannot delete files:
   - The file may be in use by another program; close it and retry

8. Updates and Data
1) App settings are saved in the user profile directory and persist after restart.
2) It is recommended to back up custom tracks in the music folder regularly.

Enjoy using Ameath Desktop Pet!
